# AdCraft v2

AdCraft v2 is an autonomous marketing platform controlled by natural language.

**`_legacy-v1-python/`** contains the v1 Python backend, archived for reference only. It is not active code — v2 runs entirely on `workers/` (Node.js TypeScript) + `app/` (Next.js). See `MIGRATION_GUIDE.md` for what is still worth porting.

For architecture overview, see `PRD.md` and `MIGRATION_GUIDE.md`.
